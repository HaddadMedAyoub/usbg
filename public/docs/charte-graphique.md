# Charte Graphique — USBG 90

## 1) Identité visuelle
- Style: sportif, énergique, moderne, mobile-first
- Priorité: lisibilité sur mobile + contraste fort

## 2) Palette de couleurs (extrait du logo)
### Couleurs principales
- **USBG Yellow (Primary):** `#FEDA01`
- **USBG Black (Primary):** `#0B0B0B`

### Couleurs secondaires
- **Gold Dark (Secondary):** `#CBAE01` (pour hover, séparateurs, badges)
- **Deep Black:** `#000000` (fonds très sombres)
- **Olive Dark:** `#3E3500` (ombres/accents discrets sur fond jaune)

### Neutres (UI)
- **Text on light:** `#111827`
- **Border light:** `#E5E7EB`
- **Background light:** `#FFFFFF`
- **Muted text:** `#6B7280`

## 3) Règles d’utilisation (important)
- Fond principal recommandé: **blanc** ou **noir**
- Texte sur fond jaune: utiliser **noir** (`#0B0B0B`) pour le contraste
- Texte sur fond noir: utiliser **jaune** (`#FEDA01`) ou blanc
- Ne pas utiliser des jaunes “néon” différents: garder le jaune du logo

## 4) Typographie (mobile-first + arabe)
Recommandation:
- **Titres:** "Cairo" (ou "Tajawal") — Bold
- **Texte:** "Cairo" Regular / Medium
- **Chiffres / stats:** "Cairo" SemiBold

Règles:
- Taille min texte mobile: **16px**
- Interligne: **1.5** pour les paragraphes
- Toujours respecter RTL pour l’arabe (dir="rtl")

## 5) Grille / Layout (mobile-first)
- Max width contenu: **1024px** (desktop), mais priorité mobile
- Padding mobile: **16px** (px-4)
- Sections: cartes (cards) avec arrondi 16–24px

## 6) UI Components (recommandations)
### Boutons
- Primary: fond jaune + texte noir
- Secondary: fond noir + texte jaune
- Ghost: transparent + bordure grise

### Cartes (News / Players)
- Fond blanc
- Bordure légère
- Ombre douce (optionnel)
- Image 16:9 (cover) pour les news

## 7) Iconographie & images
- Icônes simples (stroke)
- Photos: contraste élevé, éviter filtres trop forts
- Toujours garder un espace autour du logo (padding)

## 8) Accessibilité
- Contraste fort: jaune/noir OK
- Ne pas mettre du texte gris clair sur jaune
- Cibles tactiles mobile: min 44px de hauteur
